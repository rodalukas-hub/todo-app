# Todo-app

En fullständig todo-applikation byggd med React, TypeScript och Vite.

## Funktioner

- Lägg till och ta bort uppgifter
- Markera uppgifter som klara
- Filtrera: alla, aktiva, klara
- Rensa alla klara uppgifter
- Automatisk sparning i `localStorage`

## Tekniker

- React 19
- TypeScript
- Vite
- Tailwind CSS

## Kom igång

```bash
npm install
npm run dev
```

Öppna [http://localhost:5173](http://localhost:5173).

## Bygg för produktion

```bash
npm run build
```
